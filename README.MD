### exporting svg components.

copy svg image to `/public/icons` and run command:

```bash
    yarn svgr
```

### Installing all package.

```bash
    yarn install
```

### Running the development server.

```bash
    yarn run dev
```

### Building for production.

```bash
    yarn build
```

### Running the production server.

```bash
    yarn start
```
